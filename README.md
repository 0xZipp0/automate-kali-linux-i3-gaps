```bash
create a root user
sudo su 
passwd

```
2.clone my Repo;
```bash
git clone https://github.com/a0x14D/kali-i3wm.git
```
3.cd && give execute file
```bash
  cd kali-i3wm && chmod +x set-up.sh install.sh 
```
4.setUP
```bash
./set-up.sh
```
5.install

After complete set-up.sh 
```bash
./install.sh
```
